About the Project

About the Dataset

This dataset has been augmented offline from the original dataset to increase its size and variability. The original dataset can be found on the New Plant Disease Dataset page on Kaggle. This augmented dataset consists of approximately 87,000 RGB images of healthy and diseased crop leaves, categorized into 38 different classes.

Dataset Structure

  • Training set: 70,295 images
  • Validation set: 17,572 images
  • Test set: 33 images

The dataset is a valuable resource for training and evaluating models for plant disease detection, providing a diverse set of images to ensure robust model performance.
